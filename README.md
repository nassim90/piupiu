# piupiu

Projet 5 OpenClassRooms

## Contrainte technique
* symfony 3

## Guide d'installation
  * `git clone git@github.com:OCR-P5/piupiu.git InstallDirectory`
  * `cd InstallDirectory`
  * `composer update`
  * Provide parameters or change `parameters.yml` later
  * Create database
    * `php/bin console d:d:c`
